My UX Skills

User Research

I begin the design process with qualitative research in order to understand user pain points and needs. I then synthesize my research into an empathy map in order to discover important user insights.

User Interface Design

Once I've defined the problem I wireframe the solutions I come up with and then test the prototype. I then create a styleguide in order to define the brand. The main tools I use are Adobe XD and InVision.

Web Development
